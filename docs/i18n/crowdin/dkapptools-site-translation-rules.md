# DK App Tools Bulgarian Translation Rules

These notes are for manual Bulgarian translation of DK App Tools website copy.

## Do not translate

- DK App Tools
- Offline FileSecure
- Google Play
- Android
- Google Play Store
- Cloudflare
- GitHub
- AES-GCM
- PIN
- URL

## Preferred Bulgarian terminology

- Vault -> сейф
- Vault PIN -> PIN за сейфа
- Recovery key -> ключ за възстановяване
- Protected files -> защитени файлове
- Privacy Policy -> Политика за поверителност
- Support -> Поддръжка
- Import -> добавяне
- Export -> извеждане
- No account -> без акаунт
- No cloud upload -> без качване в облак
- No ads -> без реклами
- No analytics -> без аналитични инструменти
- No tracking -> без проследяване
- App updates -> актуализации на приложението

## Tone

Bulgarian should be natural, professional, clear, and trustworthy.

Avoid literal machine translation. Avoid exaggerated marketing. Privacy text must be formal and precise. Product text may be warmer, but must stay accurate.

## Security/privacy claims

Preserve the meaning:

- No account.
- No cloud upload.
- No ads.
- No analytics.
- No tracking.
- Protected files are not uploaded to DK App Tools.
- Google Play may be used to check for and install app updates.
- DK App Tools cannot remotely recover vault files.

Do not add stronger claims.

Do not use phrases like "100% secure", "impossible to access", "military-grade", or "guaranteed protection".
